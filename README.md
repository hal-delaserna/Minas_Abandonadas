# Minas_Abandonadas

---

Objetivo: Identificar minas que foram abandonadas por seus titulares usando bases de dados abertos. 
Essa é uma apuração usualmente feita por constatação *in loco*, o que nem sempre é possível ou desejável (consumo de recursos e custo de oportunidade). Ademais, a inspeção presencial nem sempre será taxativa: a distinção entre uma longa paralisação e um verdadeiro abandono pode não ser óbvia.

Em invés disso, nossa alternativa metodológica é **inferir abandono da área pelo abandono do processo**. Ou seja, assumir que uma área de lavra foi abandonada quando o processo minerário ao qual ela se vincula, também foi abandonado. Para isso, recorremos aos dados abertos da Receita Federal. Processos minerários ativos, mas vinculados a CNPJs extintos, não têm titularidade identificável e podem ser considerados abandonados.
