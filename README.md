# Minas_Abandonadas

---

Objetivo: Identificar minas que foram abandonadas usando dados abertos. 
O abandono de mina é usualmente constatado *in loco* o que nem sempre é possível ou desejável (consumo de recursos e custo de oportunidade). Ademais, a inspeção presencial nem sempre é conclusiva. A distinção entre uma longa paralisação e um verdadeiro abandono pode não ser óbvia.
Um expediente metodológico é **inferir abandono da área pelo abandono do processo**. Ou seja, assumir que uma mina foi abandonada quando o processo minerário ao qual ela se vincula também foi abandonado. Para isso recorremos aos dados abertos da Receita Federal e a situação do CNPJ.
